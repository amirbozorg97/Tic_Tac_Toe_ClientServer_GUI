-Download the serer jar file from discussion board(d2l).

-Double click on the jar file.

-It shows you that server is up and running.

-When the first player connects to the server, you will see a message on the text area which show the time in which first player connected to the server.

-Then, the server will wait for the second player. It’ll display the second player’s connections as well as first client.

-Clear button will clear the text area.

-Disconnect button will disconnect the server from players and exit the window.
